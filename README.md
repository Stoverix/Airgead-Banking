# Airgead-Banking
## Project Summary: The project I chose for my portfolio is the Airgead Banking application. This helps users see their investments grow over time, whether or not they make monthly deposits. It compares long-term financial growth using different savings strategies. The program gathers user input, including initial investment, monthly deposit, interest rate, and investment duration. It then calculates and shows yearly balances and interest earned in a clear report.
## What Went Well: A strength I demonstrated was clearly structuring the program using multiple files and a class. I also designed clear output tables that show annual investment growth, enhancing the user experience and making the results easy to interpret.
## Enhance Implementing: stricter input validation, developing reusable functions, and supporting various compounding periods beyond yearly calculations would boost the program's efficiency, security, and adaptability.
## Most Challenging: The toughest aspect was accurately implementing financial calculations and ensuring the results matched expectations for both scenarios. I addressed these by dividing the problem into smaller parts, testing each calculation, and using debugging to identify errors and improve formatting.
## Transferable Skills: This project helped me develop skills for future projects. I improved my understanding of object-oriented programming, especially using classes. Additionally, I enhanced my problem-solving and debugging skills through calculation and formatting challenges.
## Maintainable, Readable, Adaptable: I improved the program's maintainability and readability by using descriptive variable and function names, splitting the code into separate files, and ensuring each function handles only one task. I also included comments to clarify key parts of the logic. This approach makes the program simpler to understand and easier to update later.
